# OnlineSaha
(EN)

This repository contains the public-facing skeleton of the OnlineSaha project.

OnlineSaha is a modern platform that enables users across Türkiye to find teams for football (halı saha) matches based on their location, offering real-time opponent matching and team formation features.

Project Goals

-Help users quickly find teammates or opponents for halı saha matches

-Provide location-based automatic opponent matching

-Deliver a modern system compatible with both Web and Mobile (Android)

-Implement real-time communication and matching infrastructure
